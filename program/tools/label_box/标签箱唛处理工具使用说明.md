# 标签箱唛处理工具使用说明

## 功能概览

本工具用于根据 Excel 订单数据，自动生成打印用的**商品标签**和**外箱箱唛**文件，支持 3C 标签和玩具标签两种类型，并可选择输出内容和打包 ZIP。

## 典型使用流程

1. **选择 Excel 文件**
   - 在首页点击“标签箱唛处理工具”卡片。
   - 在“选择Excel文件”卡片中，点击 `选择 Excel 文件`，选中包含订单数据的 Excel。

2. **设置处理选项**（处理选项卡片）
   - `类型识别：`
     - `自动`：根据数据自动判断 3C / 玩具。
     - `3C`：强制按 3C 标签规则处理。
     - `玩具`：强制按玩具标签规则处理。
   - `输出内容：`
     - `标签+箱唛`：同时生成标签和箱唛文件。
     - `仅标签`：只生成标签文件。
     - `仅箱唛`：只生成箱唛文件。
     - `仅预定表`：只生成内部预览/对账表。
   - 其他按钮：
     - `打包ZIP`：将生成的文件统一打成一个 ZIP 包。
     - `生成日志文件`：输出处理日志，便于排查问题。

3. **开始处理**
   - 配置好选项后，点击 `开始处理` 按钮（如界面中提供）。
   - 程序会在输出目录中生成对应的标签/箱唛文件和预定表。

4. **查看输出结果**
   - 输出目录可通过界面右上角齿轮按钮（⚙）设置或查看。
   - 生成文件包括：
     - 标签文件（通常为图片或可打印格式）。
     - 箱唛文件。
     - 预定表/汇总表。

## 注意事项

- 建议使用标准模板的 Excel，保证列名与工具预期一致。
- 如遇行数较多，处理时间会稍长，可耐心等待。
- 遇到报错时，可以点击界面右上角的 `?` 查看本说明，或查看生成的日志文件寻求线索。
