# Razer风格设计说明

## 🎨 设计理念

Workit 已全面升级为**Razer风格**的专业黑绿配色方案，采用**黑灰为主，绿色为辅**的设计哲学，营造出高端、专业、科技感十足的视觉体验。

## 🌈 配色方案

### 主色调（黑灰系）
- **纯黑背景** `#0d0d0d` - 应用主背景，深邃稳重
- **深灰黑** `#1a1a1a` - 主要内容区域
- **中灰黑** `#2a2a2a` - 卡片和容器背景
- **浅灰黑** `#333333` - 悬停和抬高效果

### 辅助色（Razer绿）
- **标准绿** `#44d62c` - 主要交互元素（按钮、链接）
- **悬停绿** `#5ae639` - 鼠标悬停效果
- **深绿** `#2fb81b` - 按下/激活状态
- **发光绿** `#44d62c80` - 光晕效果（半透明）

### 文本颜色
- **主文本** `#ffffff` - 纯白，清晰易读
- **次要文本** `#b0b0b0` - 灰白，层次分明
- **禁用文本** `#4d4d4d` - 深灰，明确状态
- **提示文本** `#555555` - 中灰，适度弱化

## ✨ 视觉特效

### 1. 立体按钮
- **渐变背景**：使用顶部高光和底部阴影模拟3D效果
- **多层阴影**：按钮下方添加柔和阴影，增强深度感
- **按压反馈**：点击时轻微下沉，提供真实的触感反馈
- **绿色光晕**：主按钮悬停时外围发光，突出交互区域

### 2. 精致卡片
- **边框效果**：1px精细边框，清晰界定区域
- **阴影层次**：使用多层阴影营造浮起效果
- **悬停响应**：鼠标悬停时边框变为Razer绿色

### 3. 专业输入框
- **聚焦效果**：获得焦点时边框变为Razer绿色
- **占位符动画**：智能显示/隐藏占位符文本
- **内阴影**：输入框内部略微凹陷，增强真实感

## 📐 布局优化

### 间距系统
- **极小间距** `4px` - 紧密相关元素
- **小间距** `8px` - 一般元素
- **中间距** `16px` - 分组间距
- **大间距** `24px` - 区块间距
- **超大间距** `40px-60px` - 主要区域

### 内边距标准
- **按钮内边距** `24px × 12px` - 舒适的点击区域
- **卡片内边距** `20px-30px` - 宽松的内容空间
- **输入框内边距** `12px × 8px` - 易读的输入区域

### 字体层级
- **超大标题** `32px` - Hero区域
- **大标题** `24px` - 页面标题
- **中标题** `18px` - 区块标题
- **小标题** `14px` - 卡片标题
- **正文** `10px` - 常规内容
- **小字** `9px` - 辅助信息

## 🎯 交互设计

### 鼠标状态
1. **默认状态** - 黑灰背景，白色文本
2. **悬停状态** - 背景提亮，Razer绿色边框/高光
3. **按下状态** - 轻微下沉，颜色变深
4. **禁用状态** - 灰色显示，无交互

### 动画效果
- **过渡时间** - 150ms快速响应
- **缓动函数** - ease-in-out自然过渡
- **光晕效果** - 渐变出现/消失

## 🌓 深色/浅色模式

应用会自动检测系统主题设置：
- **深色模式**：黑灰背景 + Razer绿强调
- **浅色模式**：白色背景 + Razer绿强调

用户无需手动切换，应用启动时自动适配系统偏好。

## 🚀 使用方法

### 在新工具中应用Razer风格

```python
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).parent.parent.parent))
from theme import get_colors
from ui_components import RazerButton, RazerCard, RazerInput

# 获取配色方案
colors = get_colors()

# 设置窗口背景
root.configure(bg=colors['bg_main'])

# 使用Razer风格按钮
btn = RazerButton(
    parent,
    text="开始处理",
    command=your_function,
    style="primary",  # 或 "secondary"
    width=140,
    height=45
)

# 使用Razer风格卡片
card = RazerCard(parent, elevated=True)
content = card.get_content_frame()

# 使用Razer风格输入框
input_widget = RazerInput(parent, placeholder="输入内容...")
```
