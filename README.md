# Centisky

工具集合 - 图片处理、视频处理、标签生成、开票处理、数据分析

## 快速开始

```bash
python d:\Centisky\program\launcher.py
```

## 工具

- 🖼️ **图片处理**: 格式转换、拼长图、切片等
- 🎬 **视频处理**: 格式转换、压缩、调整尺寸等
- 📦 **标签箱唛**: 自动生成标签和箱唛文件
- 📄 **开票信息**: 整理开票文件，生成财务表格
- 📊 **数据分析**: 快车投流周对比分析

## 使用

1. 启动应用
2. 点击左侧边栏选择工具
3. 在右侧面板中使用工具
4. 点击顶部按钮切换主题

## 特性

- 单窗口设计
- 快速工具切换
- 深色/浅色主题
- Razer 风格设计
