# Centisky v1.1.0

集成工具启动器 - 图片处理、视频处理、标签生成、发票处理、数据分析

## 🚀 快速开始

### 使用安装程序（推荐）
下载最新版本：[Centisky-Setup-1.1.0.exe](https://github.com/Benji-Xu/Centisky/releases/tag/v1.1.0)

### 从源代码运行
```bash
python d:\Centisky\program\launcher.py
```

## 🛠️ 工具

- 🖼️ **图片处理工具**: 格式转换、拼长图、切片、抠图等
- 🎬 **视频处理工具**: 格式转换、压缩、调整尺寸等
- 📦 **标签箱唛工具**: 自动生成标签和箱唛文件
- 📄 **发票整理工具**: 整理开票文件，生成财务表格
- 📊 **京准通数据分析**: 快车投流周对比分析

## 📖 使用方法

1. 启动应用
2. 点击左侧边栏选择工具
3. 在右侧面板中使用工具
4. 点击顶部按钮切换主题（深色/浅色）

## ✨ 特性

- 单窗口集成设计
- 快速工具切换（< 0.5秒）
- 深色/浅色主题自适应
- iOS 18 风格设计
- 统一的按钮样式和间距
- 固定窗口大小（1200x700）
- 现代化UI组件

## 📋 系统要求

- Windows 10/11 (64-bit)
- Python 3.7+ (如果从源代码运行)
